## `Protfolio` Web Application

### The React Framework for the Web (Next.js 14)

### Getting Started

Clone repository:
`git clone <url>`
`cd <project-name>`

Install project dependecy:
`npm install`

Node version (18):
`node -v 18.16.0`
`nvm use 18.16.0`

Run application:
`npm run dev`

Open with your browser to see the result:
[`http://localhost:3000`](http://localhost:3000)

Installed package:
